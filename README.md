# Library Management System

კონსოლზე დაფუძნებული ბიბლიოთეკის მართვის სისტემა Python-ზე, რომელიც გულისხმობს მომხმარებლების რეგისტრაციას და მათ მიერ წიგნების გადატანა/დაბრუნების სისტემას.

---

## ძირითადი ფუნქციონალი
ლოგიკა დაყოფილია კლასების მიხედვით

  ### მომხმარებელი (User)
  - რეგისტრაცია
  - ავტორიზაცია
  - პირადი გგვერდის გაუქმება
  - პირადი გვერდის ნახვა
  - წიგნების ძებნა (სახელით / ავტორით)
  - წიგნის გატანა ვადით
  - წიგნის დაბრუნება
  - წიგნის შეფასება (0–5 შკალაში)
  
  ### ადმინისტრატორი (Admin)
  - ავტომატურად იქმნება სისტემის გაშვებისას
  - წიგნის დამატება
  - წიგნის წაშლა
  - ყველა წიგნის ნახვა
  
  ### წიგნები
  - ინახება JSON ფაილში
  - რეიტინგების დაგროვება
  - საშუალო რეიტინგის ავტომატური გადათვლა
  - რეიტინგის მიხედვით სორტირება

## არქიტექტურული პრინციპები

- UI და ლოგიკა განცალკევებულია
- Library კლასი ცალკეა გამოყოფილი ბიბლიოთეკის შესაბამისი ფუნქციებისთვის და არ შეიცავს UI კოდს
- LibraryApp კლასი ცალკეა გამოყოფილი UI-სთვის და არ აქვს კვეთა ლოგიკურ ნაწილთან
- ყველა ვალიდაცია ერთიანდება `validators` კლასში


## მონაცემთა შენახვა

- მონაცემები ინახება ავტომატურად ყოველი ცვლილებისას
- `data/users.json` — მომხმარებლები
- `data/books.json` — წიგნები

## პროექტის მიმდინარეობა
- პროექტის თითოეული მონაწილე მუშაობდა შესაბამის ბრენჩზე
- კოდის ცვლილებები რეგულარულად ინახებოდა და იტვირთებოდა ბრენჩებზე
- ვერსიების დასინქრონება ხდებოდა ბრენჩების დამერჯვით

- ლიზი კენჭოშვილი
